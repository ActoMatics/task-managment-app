## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Prerequisite
Nodejs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API documentations
Run - npm run apidocs
This will generate a folder called "docs" in root -> inside of it there an index.html, you can open this in the browser and get an HTML readable documentation.
Do note this folder is in .gitignore
